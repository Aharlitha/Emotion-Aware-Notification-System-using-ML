# Emotion-Aware Notification System using ML

## Overview
The Emotion-Aware Notification System is a machine learning-based application that detects a user's emotional state through speech and facial expressions and generates context-aware notifications accordingly.

## Features
- Detects emotions using facial expressions
- Analyzes speech-based emotional cues
- Multimodal emotion recognition using fusion techniques
- Generates personalized notifications based on detected emotions
- Uses machine learning models for emotion classification
- Interactive GUI support

## Technologies Used
- Python
- Machine Learning
- OpenCV
- Speech Processing
- TensorFlow
- NumPy
- GUI Modules

## Project Structure
- `vision/` → facial emotion detection modules
- `speech/` → speech emotion analysis modules
- `fusion/` → combines multiple emotion sources
- `models/` → trained ML models
- `gui/` → user interface files

## Workflow
1. Capture facial and speech inputs
2. Extract features
3. Predict emotional state
4. Fuse predictions
5. Generate suitable notifications

## Applications
- Mental wellness assistance
- Smart notification systems
- Human-computer interaction
- Personalized user experiences
